#  Studio Namma Website UI Clone

## Project Overview

The **Studio Namma Website UI Clone** is a front-end web development project built using **HTML5, CSS3, and JavaScript**. The primary objective of this project is to recreate the modern and visually engaging interface of the official Studio Namma website.

This project focuses on replicating advanced UI interactions, smooth animations, custom cursor effects, hover animations, video backgrounds, and responsive layouts without any backend integration.

It demonstrates the ability to build interactive and premium-quality web interfaces using core frontend technologies.

The clone closely resembles the original Studio Namma website in terms of layout, typography, animations, transitions, and overall user experience.

The primary purpose of this project is to strengthen frontend development skills by:

#  Installation and Execution Steps

## Step 1: Clone the repository

```bash

```

## Step 2: Navigate to the project folder

